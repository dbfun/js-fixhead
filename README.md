# JQuery FixHead

FixHead is a jQuery plugin that fix a head section of a table.

File structure:

  * `css/` - compiled CCS from `src-css`
  * `js/` - javascript plugin + demo script
  * `demo.md` - demo code

Use:

    $('.js-fixhead').fixhead();

[Live demo with examples](http://js.pushorigin.ru/fixhead)
